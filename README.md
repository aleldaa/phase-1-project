## The Filminator

### Page Layout
- View list of the top 10 movies
- Click on a movie to get the movies details
  - Movie title
  - Movie poster
  - Year Released
  - Main crew
  - imDb rating
- View rating of the movie and add your own rating
- Add a movie to the list of movies
